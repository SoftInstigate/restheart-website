== Getting Started

Sophia is an intelligent AI assistant designed to provide helpful information and support based on your organization's knowledge base. This guide will help you make the most of your interactions.

== Accessing Sophia

=== Web Interface
Access Sophia through a web browser at the URL provided by your administrator. The interface is responsive and works on desktop, tablet, and mobile devices.

=== Embedded Integration
Sophia can be embedded in other websites or applications through iframe integration.

== Using the Chat Interface

=== Starting a Conversation

Navigate to the Sophia web interface. The text input field is at the bottom of the screen — type your question in natural language and press Enter or click the send button.

image::../../../images/sophia/chat-interface.png[Chat interface, title="Sophia chat interface"]

=== Real-time Responses
Sophia streams responses in real time — you can read the answer as it's being composed. A typing indicator shows when Sophia is processing your request.

=== Agentic Responses

Some knowledge bases have *Agentic Mode* enabled. When you ask a complex question in an agentic context, Sophia doesn't answer immediately. Instead, it first searches the knowledge base, reads relevant documents, and reasons about the results — potentially performing multiple rounds of research — before composing a comprehensive response.

image::../../../images/sophia/agentic-response.png[Agentic response, title="Sophia showing agentic reasoning with tool execution badges"]

During this agentic phase, you'll see real-time badges showing what Sophia is doing:

- *Tool start*: Shows the tool name and query (e.g. "Searching for: RESTHeart configuration")
- *Tool result*: Shows the outcome and duration (e.g. "Found 5 relevant segments in 342ms")

Once the research phase is complete, the text response streams as usual.

=== Message History
Previous messages are displayed above the current conversation. Scroll up to review earlier questions and responses.

=== Message Formatting
Sophia formats responses with:

- *Bold* and _italic_ text for emphasis
- `Code formatting` for technical terms
- Code blocks with syntax highlighting
- Bulleted and numbered lists
- Links to related resources

== Customization

=== Language Selection

Sophia supports English and Italian. Set the language via URL parameter:

- `?lang=en` — English (default)
- `?lang=it` — Italian

Example: `https://your-domain.com/sophia?lang=it`

=== Theme Selection

- `?theme=default` — dark theme (default)
- `?theme=light` — light theme

Example: `https://your-domain.com/sophia?theme=light`

=== Display Modes

- `?h=f` — full height (uses entire screen)
- `?h=auto` — auto height (adjusts to content, ideal for embedded use)

=== Combining Parameters

Parameters can be combined: `https://your-domain.com/sophia?lang=it&theme=light&h=auto`

== Best Practices

=== Writing Effective Prompts

- *Be specific*: "What are the key features of RESTHeart?" rather than "Tell me about products"
- *Provide context*: mention relevant background and the type of answer you need
- *Use clear language*: write in complete sentences, avoid ambiguous terms
- *Ask one question at a time*: for complex topics, break into focused questions

=== Getting Better Results

- Ask follow-up questions for clarification or more details
- Reference earlier parts of the conversation to build context
- If a response is too broad, ask for a more specific or technical explanation

=== Understanding Limitations

- Sophia answers based on the uploaded knowledge base — it may not have information on topics outside that scope
- For critical decisions, verify information from authoritative sources
- Sophia cannot access external websites or real-time data

== Troubleshooting

=== Connection Problems
*Symptom*: Chat interface doesn't load +
*Solution*: Check your internet connection and refresh the page

=== Slow Responses
*Symptom*: Sophia takes a long time to respond +
*Solution*: Complex queries or agentic mode require more processing time. In agentic mode, you can see the progress through tool execution badges.

=== Incomplete Responses
*Symptom*: Response cuts off unexpectedly +
*Solution*: Rephrase your question or break it into smaller parts

=== Authentication Errors
*Symptom*: "Authentication failed" message +
*Solution*: Contact your administrator for access credentials

== Privacy and Data Handling

- All communications are encrypted in transit
- Access is controlled through authentication
- Conversations are not used to train the AI model
- Don't share sensitive personal information (passwords, financial details) in conversations
