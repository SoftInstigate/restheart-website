=== Overview

Sophia is exclusively available as a managed service through *RESTHeart Cloud* in the *dedicated* tier. As a fully managed service, the SoftInstigate team handles the infrastructure setup — you don't need to install or configure any software.

=== Prerequisites

- *MongoDB on Atlas* — for document storage and vector search
- *An AWS account* — for Bedrock AI model access (Claude and Titan Embed)

=== Getting Started

Contact link:mailto:support@restheart.com[support@restheart.com] to request a dedicated service with the Sophia plugin.

The SoftInstigate team will guide you through:

. *AWS Bedrock configuration* — creating an IAM user with the required Bedrock and Marketplace permissions and generating access keys
. *MongoDB Atlas vector index* — setting up the vector search index on the `textSegments` collection for semantic search
. *Initial context setup* — creating your first context with a prompt template and knowledge base tags

=== After Setup

Once the service is provisioned, you can:

- Log in to the *Admin Panel* to manage contexts, upload documents, and issue API tokens — see the link:/docs/cloud/sophia/administrator-guide[Administrator Guide]
- Share the chat URL with your users — see the link:/docs/cloud/sophia/user-guide[User Guide]
- Connect AI clients via MCP — see link:/docs/cloud/sophia/mcp[Sophia MCP Server]
