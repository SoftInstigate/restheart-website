== Sophia MCP Server

Sophia exposes a link:https://modelcontextprotocol.io[Model Context Protocol (MCP)] server, so any MCP-compatible AI client (Claude Desktop, Cursor, VS Code, Claude Code…) can query your knowledge base directly.

=== Endpoint

[source]
----
https://sophia-api.restheart.com/mcp
----

No authentication required.

=== Quick Setup

Pick your client, then paste the snippet into your config file.

NOTE: If your client supports only `stdio` (e.g. older Claude Desktop), use the `mcp-remote` variant — it requires link:https://nodejs.org[Node.js ≥ 18].

==== Claude Desktop

Config file location:

* *macOS:* `~/Library/Application Support/Claude/claude_desktop_config.json`
* *Windows:* `%APPDATA%\Claude\claude_desktop_config.json`

*RESTHeart*

[source,json]
----
{
  "mcpServers": {
    "sophia": {
      "command": "npx",
      "args": [
        "mcp-remote",
        "https://sophia-api.restheart.com/mcp",
        "--header", "X-Sophia-Tags: restheart"
      ],
      "env": {}
    }
  }
}
----

*RESTHeart Cloud*

[source,json]
----
{
  "mcpServers": {
    "sophia": {
      "command": "npx",
      "args": [
        "mcp-remote",
        "https://sophia-api.restheart.com/mcp",
        "--header", "X-Sophia-Tags: cloud",
        "--header", "X-Sophia-Prompt: cloud"
      ],
      "env": {}
    }
  }
}
----

Restart Claude Desktop. Sophia will appear as a connected tool.

----

  "sophia": {

    "url": "https://sophia-api.restheart.com/mcp",
    "headers": {
      "X-Sophia-Tags": "restheart"
    }
  }
}

[source,json]

=== Optional Headers

[cols="1,2", options="header"]
|===
| Header | Description
| `X-Sophia-Tags` | Filter documentation by tag: `restheart` for on-prem docs, `cloud` for RESTHeart Cloud docs. Omit to search all documents.
| `X-Sophia-Prompt` | Select a prompt template tuned for the target product. Omit to use the default.
|===

=== Available Tools

[cols="1,2", options="header"]
|===
| Tool | Description
| `sophia_search` | Semantic search over the RESTHeart knowledge base. Returns relevant text segments.
| `sophia_render_prompt` | Builds a fully interpolated prompt with RAG context, ready to send to any LLM.
| `sophia_api` | Returns the complete Sophia REST API reference. Requires authentication — reserved for service administrators.
|===
